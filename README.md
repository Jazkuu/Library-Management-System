# Library Management System

## Description

This project is a **Library Management System** built in Python using **Object-Oriented Programming (OOP)**.

It allows users to manage books and members, as well as issue and return books through a "simple" commandline interface.

---

## Features

### Book Management

* Add new books
* Remove books
* Update book details
* Display all books

### Member Management

* Add members
* Remove members
* Update member details
* Display all members

### Book Transactions

* Issue books to members
* Return books
* Error handling

---

## OOP Concepts Used

* **Classes And Objects**
* **Encapsulation**
* **Inheritance**
* **Polymorphism**
